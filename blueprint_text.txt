 1. Project Overview & Architecture
 Platform Vision: CareBridge Health is a comprehensive Healthcare Referral Management SaaS platform designed to bridge the gap between doctors (consultants), hospitals (IPD), laboratories, and patients in Pakistan. The platform enables seamless patient referrals, real-time bed management, commission tracking, and emergency escalation.

 1.4 Key Business Rules (From Original Portal Document)
 Critical Business Logic:
 System-generated first password for all users (Doctor, Hospital, Admin)
 One doctor can create multiple portal IDs with different usernames/passwords
 Patient records are end-to-end encrypted between Doctor, IPD, and Admin
 Patient Guardian ID and CNIC required for tracking
 Unique serial ID generated for every patient
 Real-time updates shared between Doctor portal and IPD portal immediately upon reservation
 Admin CANNOT interfere in treatment activity - only monitoring and authorization
 Commission records display ONLY in admin portal
 Hospital pays doctor through bank account; platform commission recorded separately
 Platform bank account API connected in Admin portal

 6.3 Emergency Escalation Rules (Auto-Triggered)
RULE 1: IF no hospital response in 3 minutes → Expand to Top 10 hospitals
RULE 2: IF Emergency priority detected → Auto-assign nearest facility with capacity
RULE 3: IF ICU bed unavailable → Search HDU / NICU / PICU alternatives
RULE 4: IF all beds full in network → Escalate to network admin + SMS alert
RULE 5: IF SLA timer expires → Auto-escalate + mark as critical
